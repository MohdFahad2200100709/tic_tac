1.Tic Tac Toe Game

  This is a web-based Tic Tac Toe game created using HTML, CSS, and JavaScript.

2.Technologies Used
  - HTML
  - CSS
  - JavaScript

3.Features
  - Two-player game
  - Responsive design
  - Win and draw detection
  - Restart game button

4. How to Run
  - Download or clone the repository
  - Open `index.html` in any web browser
  - Play the game

5. How to Play
  - Player 1 uses **X**
  - Player 2 uses **O**
- Players take turns clicking on the grid
- First player to get 3 marks in a row wins
